Introdução ao HTML

Malkdev0
JMFXP22

HTML é a espinha dorsal de qualquer página web. Ele define a estrutura e o conteúdo da página.

O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).

Não é uma linguagem de programação, mas sim uma linguagem de marcação que define a estrutura do seu conteúdo.

Elementos HTML

Os elementos HTML são os blocos de construção de uma página HTML. Eles são representados por tags.

<tagname>Conteúdo</tagname>

Exemplo de elementos HTML comuns:

  • <h1> a <h6> para títulos
  • <p> para parágrafos
  • <a> para links
  • <img> para imagens
Para fixar na mente: Escreva as tags mais importantes em seu caderno e revise diariamente por uma semana.

Estrutura Básica

Toda página HTML possui uma estrutura básica que inclui declaração DOCTYPE, html, head e body.

Estrutura Completa

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Página</title>
</head>
<body>
  <h1>Meu Primeiro Cabeçalho</h1>
  <p>Meu primeiro parágrafo.</p>
</body>
</html>

Partes da Estrutura

  • <!DOCTYPE html> - Define que este documento é HTML5
  • <html> - O elemento raiz de uma página HTML
  • <head> - Contém metainformações sobre o documento
  • <title> - Especifica um título para o documento
  • <body> - Contém o conteúdo visível da página

Variáveis são containers para armazenar dados. Java é uma linguagem fortemente tipada, o que significa que você deve declarar o tipo de cada variável.

Declarando Variáveis

tipo nomeDaVariavel = valor;

Exemplos:

Tipos de Dados Primitivos

  • byte - 8 bits (-128 a 127)
  • short - 16 bits (-32,768 a 32,767)
  • int - 32 bits (-2^31 a 2^31-1)
  • long - 64 bits (-2^63 a 2^63-1)
  • float - Ponto flutuante de 32 bits
  • double - Ponto flutuante de 64 bits
  • boolean - true ou false
  • char - Um único caractere Unicode (16 bits)

Variáveis e Tipos

Malkdev0
JMFXP22

Lua é uma linguagem dinamicamente tipada, o que significa que as variáveis não têm tipos; os valores têm.

Declarando Variáveis

Em Lua, você não precisa declarar variáveis com um tipo específico. Basta atribuir um valor:

nome = "Maria" -- string
idade = 25 -- number
ativo = true -- boolean

Tipos de Dados em Lua

  • nil - Representa a ausência de um valor útil
  • boolean - true ou false
  • number - Representa números reais (doubles)
  • string - Sequência de caracteres
  • function - Funções
  • table - Estrutura de dados principal (arrays, dicionários, etc.)

Verificando Tipos

Use a função type() para verificar o tipo de um valor:

print(type("Hello")) -- string
print(type(10.4)) -- number
print(type(print)) -- function
print(type(true)) -- boolean
print(type(nil)) -- nil

Lua oferece estruturas de controle familiares para gerenciar o fluxo de execução do programa.

Condicionais: if, elseif, else

if condição then
  -- código se a condição for verdadeira
elseif outra_condição then
  -- código se a outra condição for verdadeira
else
  -- código se nenhuma condição for verdadeira
end

Loops

while loop:

while condição do
  -- código a ser repetido
end

repeat-until loop:

repeat
  -- código a ser repetido
until condição

for loop:

for variável = início, fim, incremento do
  -- código a ser repetido
end

Exemplo Prático

-- Imprimir números de 1 a 5
for i = 1, 5 do
  print(i)
end

-- Contar até 10 com while
contador = 1
while contador <= 10 do
  print(contador)
  contador = contador + 1
end
