</html>

HTML é a espinha dorsal de qualquer página web. Ele define a estrutura e o conteúdo da página.

O que é HTML?

HTML significa HyperText Markup Language (Linguagem de Marcação de Hipertexto).

Não é uma linguagem de programação, mas sim uma linguagem de marcação que define a estrutura do seu conteúdo.

Elementos HTML

Os elementos HTML são os blocos de construção de uma página HTML. Eles são representados por tags.

<tagname>Conteúdo</tagname>

Exemplo de elementos HTML comuns:

  • <h1> a <h6> para títulos
  • <p> para parágrafos
  • <a> para links
  • <img> para imagens
Para fixar na mente: Escreva as tags mais importantes em seu caderno e revise diariamente por uma semana.

Estrutura Básica

JMFXP22

Toda página HTML possui uma estrutura básica que inclui declaração DOCTYPE, html, head e body.

Estrutura Completa

<!DOCTYPE html>
<html lang="pt-BR">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <title>Título da Página</title>
</head>
<body>
  <h1>Meu Primeiro Cabeçalho</h1>
  <p>Meu primeiro parágrafo.</p>
</body>
</html>

Partes da Estrutura

  • <!DOCTYPE html> - Define que este documento é HTML5
  • <html> - O elemento raiz de uma página HTML
  • <head> - Contém metainformações sobre o documento
  • <title> - Especifica um título para o documento
  • <body> - Contém o conteúdo visível da página
Para fixar na mente: Crie pelo menos 5 páginas HTML com esta estrutura básica. A prática constante é essencial!

HTML oferece várias tags para formatar texto e melhorar a legibilidade do conteúdo.

Tags de Formatação

<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<p>Este é um <strong>parágrafo</strong> com <em>ênfase</em>.</p>
<ul>
  <li>Item 1 da lista</li>
  <li>Item 2 da lista</li>
</ul>

Tags Comuns de Formatação

  • <strong> - Texto importante (normalmente em negrito)
